# tree-sitter-viewer 

这是一个类似于 `Tree-sitter` 官网[`游乐场`](https://tree-sitter.github.io/tree-sitter/7-playground.html)的`VS Code`扩展插件。  
您可以使用该插件方便的查看不同开发语言的抽象语法树，插件支持的语言与 `Tree-sitter` 官网 `游乐场` 保持一致。
